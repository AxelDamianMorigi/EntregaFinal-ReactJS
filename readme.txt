
Proyecto Neo Celulares - Axel Morigi

¡Bienvenido a Neo Celulares!

Descripción
Neo Celulares es un proyecto web de comercio electrónico desarrollado con React. En esta plataforma, podrás explorar una variedad de teléfonos celulares, obtener detalles de productos, gestionar un carrito de compras y finalizar tus compras de manera sencilla.

Contenido del Repositorio
src: Contiene el código fuente del proyecto.
components: Componentes reutilizables de React.
contexto: Gestión del estado global para el carrito de compras.
firebase: Configuración y utilidades para la integración con Firebase.
pages: Páginas principales de la aplicación.
styles: Estilos globales y archivos de estilo.
App.jsx: Punto de inicio de la aplicación.

Características Principales
Exploración de Productos: Navega por la variedad de teléfonos disponibles.
Detalles del Producto: Obtén información detallada de productos al hacer clic.
Carrito de Compras: Agrega y revisa productos en tu carrito de compras.
Finalizar Compra: Completa tus compras en la página de carrito.
Estado Global del Carrito
Hemos implementado contextos para gestionar el estado global del carrito de compras, permitiendo una experiencia de compra fluida.
